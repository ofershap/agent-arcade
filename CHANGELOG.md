# Changelog

## 0.1.0

Initial release.

- Pixel art office that reacts to your Cursor AI agent's activity
- Character sits at desk and types when the agent writes code, reads files, or runs commands
- Idle behavior: coffee breaks, petting the cat, browsing bookshelf, watering the plant, playing arcade
- Interactive objects: lamp toggle, window curtains with real-time sky, arcade mini-games, clickable everything
- Click-to-attract: click any object while idle and the character walks over
- Click combos with floating text
- Plugin API for custom objects and backgrounds
- Zero config, zero images — all sprites drawn from code
